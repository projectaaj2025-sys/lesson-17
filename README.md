# 📚 Урок 17: Академическая грамотность

Интерактивный тест с 3 вариантами на тему библиографических ссылок, плагиата и научного стиля.

## 📁 Файлы проекта

```
pedagogika-test/
├── index.html              # Страница учеников (CSS + JS встроены)
├── teacher.html            # Панель учителя (CSS + JS встроены)
├── google-apps-script.js   # Код для Google Apps Script
└── README.md
```

## 🚀 Быстрый старт

### 1. Создайте Google Таблицу
- [sheets.new](https://sheets.new) → скопируйте ID из URL

### 2. Создайте Google Apps Script
- [script.google.com](https://script.google.com) → Новый проект
- Вставьте код из `google-apps-script.js`
- **Развернуть** → **Веб-приложение** → Доступ: **Все**

### 3. Загрузите на GitHub Pages
- Создайте репозиторий, загрузите `index.html` и `teacher.html`
- Settings → Pages → main branch, / (root)

## 🎮 Возможности

| Функция | Описание |
|---------|----------|
| **Случайный вариант** | 1 из 3 с равной вероятностью |
| **Соединение стрелками** | Drag-and-drop для соответствий |
| **Корзины** | Drag-and-drop для классификации |
| **Сравнение текстов** | Два текста рядом + поле для ответа о плагиате |
| **Переписывание** | Поле для перефразирования в научный стиль |
| **Заполнение пропусков** | Выпадающие списки для оформления ссылок |
| **Автопроверка** | Мгновенный результат с разбором ошибок |
| **Google Таблица** | Автосохранение всех результатов |
| **Панель учителя** | Статистика, загрузка, экспорт CSV, очистка |
